# 创建HTML交互页面

## 1. 确定项目需求

- [ ] 定义一个简单的HTML结构

## 2. 编写HTML代码

- [ ] 使用<!DOCTYPE html>声明文档类型

- [ ] 编写<html>, <head>, <body>标签

- [ ] 在<body>中添加<button>元素，并设置文本为'点击我'

- [ ] 给按钮添加id属性，值为myButton

## 3. 添加JavaScript交互逻辑

- [ ] 在<head>部分引入<script>标签

- [ ] 定义一个函数handleClick，当按钮被点击时执行

- [ ] 在<button>元素上添加onclick事件监听器，调用handleClick函数

## 4. 测试页面

- [ ] 保存文件并使用浏览器打开查看效果
